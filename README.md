WineMerge
==============================
DockerのWine上でWinMergeを起動する．

BUILD
--------------------
```
$ git clone https://github.com/HeavyMoon/WineMerge.git
$ cd WineMerge/
$ docker build -t winemerge .
```

HOW TO USE
--------------------
```
$ ./WineMerge
```
